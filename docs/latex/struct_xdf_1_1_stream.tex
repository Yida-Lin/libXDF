\doxysubsection{Detailed Description}
X\+DF files uses stream as the unit to store data. An X\+DF file usually contains multiple streams, each of which may contain one or more channels. The \mbox{\hyperlink{struct_xdf_1_1_stream}{Stream}} struct stores meta-\/data, time series, timetamps and other information of a stream.
